# Automato

Automato AFND e AFD, com fita de saída e tabela de saída.

## Uso
Se quiser testar com outros tokens/regras, altere o arquivo "entrada_automato.txt"

### Para rodar:
### Necessário java => 21.0.2
### Linux
```
javac -d ./out  ./*.java 
```
```
java -cp ./out  ./Main.java 
```
### Windows

```
javac -d .\out  *.java 
```
```
java -cp .\out .\Main.java 
```
